# shugiin-data-collector

収集結果は https://github.com/tcbn/shugiin-data からダウンロードできます。

## Build

```bash
yarn install --ignore-optional
yarn
```

## Usage
```bash
$ ./bin/shugiin gian
Usage: shugiin gian [options] [command]

議案 data subcommand

Options:
  -h, --help             display help for command

Commands:
  list                   show 回次 that Raw議案 can fetch.
  fetch <start> <end>    fetch Raw議案 from <start> 回次 to <end> 回次.
  convert <start> <end>  convert Raw議案 to 議案 from <start> 回次 to <end> 回次.
  help [command]         display help for command
```
